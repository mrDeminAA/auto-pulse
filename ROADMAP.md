# Roadmap проекта AutoPulse

Долгосрочный план развития проекта по кварталам.

---

## Q1 2026 (Февраль — Апрель)

**Тема:** Фундамент и базовая инфраструктура

### Февраль 2026
- [x] Создание структуры решения
- [x] Настройка Docker Compose (PostgreSQL, Redis, RabbitMQ, Seq)
- [ ] Настройка NuGet пакетов
- [ ] Базовые доменные сущности
- [ ] Первичный дизайн БД

### Март 2026
- [ ] Парсер США (Cars.com) — базовая версия
- [ ] Сохранение данных в PostgreSQL
- [ ] Базовое API для чтения данных
- [ ] Кэширование в Redis

### Апрель 2026
- [ ] Парсер Европы (Mobile.de)
- [ ] Аналитика: сравнение цен по регионам
- [ ] Простой Angular дашборд (1-2 графика)

**Результат квартала:** Работающий прототип с парсингом США и базовым отображением данных.

## Q2 2026 (Май — Июль)

**Тема:** Масштабирование и функциональность

### Май 2026
- [ ] Парсер Китая (Autohome)
- [ ] Парсер Кореи (Encar)
- [ ] Оптимизация парсеров (прокси, rate limiting)

### Июнь 2026
- [ ] Аналитический модуль (тренды, статистика)
- [ ] Система уведомлений (Telegram, Email)
- [ ] Поиск и фильтры по автомобилям

### Июль 2026
- [ ] Полный Angular дашборд
- [ ] Экспорт отчётов (PDF, CSV)
- [ ] Авторизация и пользователи

**Результат квартала:** Полнофункциональная система с парсингом 4 регионов и дашбордом.
